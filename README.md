# WeChat-Work-Hook 🚀

基于 Hook 技术的企业微信自动化助手框架。支持消息实时收发、群组自动化管理，为机器人开发者提供稳定、高效的接口支持。

### 🌟 核心功能
- **消息 Hook**：实时拦截、修改、发送企微各类消息（文本、图片、文件等）。
- **自动化管理**：支持自动入群、修改群公告、成员管理等。
- **低侵入性**：无需修改企微源码，稳定兼容。
